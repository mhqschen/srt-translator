# SRT 字幕翻译工具（GUI）

本仓库提供 Windows 下的 Python GUI 程序：`srt_gui.py`。

## 功能
- 字幕翻译器：AI 翻译 SRT
- 字幕校验器：检查原文/译文条目一致性与常见问题
- 双语转换器：合成双语字幕
- 字幕纠错器：针对 ASR（如 Whisper）识别错词进行纠错，并提供审核页
- 字幕润色器：对字幕进一步润色，并提供审核页

## 运行环境
- Windows 10/11
- Python 3.8+（需要包含 Tkinter；通常安装 Python 时默认自带）

## 安装依赖
在仓库目录执行：
```bat
python -m pip install -r requirements.txt
```

## 运行
二选一：

1) 直接运行：
```bat
python srt_gui.py
```

2) 使用启动脚本（推荐，包含基础环境检查）：
- 双击运行 `run_gui.bat`

## 配置文件
- 程序会在同目录读取/写入 `srt_gui_config.json`
- 该文件通常包含 API Key 等敏感信息：请自行妥善保管，避免上传到公开平台

## 参数建议
**速度 & 质量兼顾（推荐）**
- 字幕翻译器：批次大小 `30`，上下文大小 `5`，线程数 `20`，温度 `0.8`
- 字幕纠错器：批次大小 `50`，线程数 `20`，温度 `0.2`，上下文窗口 `5`，超时秒数 `300`，批量模式 `真批量`
- 字幕润色器：批次大小 `30`，上下文大小 `2`，线程数 `20`，温度 `0.6`

**更稳（网络波动/限流时）**
- 批次大小 `10`，上下文大小 `2`，线程数 `5`
