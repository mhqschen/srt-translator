SRT 字幕纠错详细指南

修复 ASR（如 Whisper）识别导致的错词/错字、标点与基础格式问题，并提供“审核页”逐条确认。

先把软件跑起来（Python 版）

  1. 安装 Python 3.8+（安装时建议勾选 “Add Python to PATH”）。
  2. 在本项目目录打开命令行，安装依赖：
    python -m pip install -r requirements.txt
  3. 启动方式二选一：
    • 双击 run_gui.bat
    • 命令行运行 python srt_gui.py

适用场景

  • 字幕来自 Whisper/剪辑软件自动识别，存在大量“同音错词/错字”。
  • 希望尽量保持原句结构，只修正“发音相近导致的识别错误”。
  • 需要在 AI 纠错后逐条审核确认，避免误改专有名词。

使用前准备

强烈建议先备份原始字幕（复制一份同目录即可）。
  • 尽量使用 UTF-8 编码的 .srt 文件。
  • 文件路径尽量避免过深目录；中文路径通常没问题。
  • 确保 API 可用（可在“字幕翻译器”页测试 API 连接）。

操作步骤（推荐流程）

1）进入“字幕纠错器”标签页

  • 选择 输入 SRT（原始识别字幕）。
  • 设置 输出 SRT（建议自动生成或手动指定新文件名）。

2）设置参数（推荐）

批次大小
50
线程数
20
温度
0.2
上下文窗口
5
超时秒数
300
批量模式
真批量
不稳定时：优先把线程数降到 5，再把批次大小降到 10。

3）开始纠错（可强制停止）

  • 点击“开始纠错”。
  • 运行中点击“停止纠错”，会以强制停止为目标终止任务。
  • 结束后会自动跳转到“纠错/润色审核”页，方便你立刻审核。

4）在“纠错/润色审核”里逐条确认

  • 你可以选择保留原文、接受纠错或手工编辑。
  • 手动编辑后记得保存修改，再导出最终结果。

常见建议

  • 纠错策略：优先纠正“发音相近导致的错误”，不要随意改写语序或语法。
  • 专有名词：遇到型号、人名、地名等，建议在审核页重点核对。
  • 稳定性：不稳定时先降并发（线程数/批次大小）。
注意：AI 纠错并不保证 100% 正确，最终以人工审核为准。
